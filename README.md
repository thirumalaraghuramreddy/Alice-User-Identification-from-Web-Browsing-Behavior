# Alice User Identification from Web Browsing Behavior

## Project Overview
This project focuses on identifying whether a web browsing session belongs to Alice or another user based on sequences of visited websites. The objective is to analyze user behavior patterns and build a machine learning model that accurately classifies browsing sessions.

## Problem Statement
Given a sequence of websites visited during a browsing session, predict whether the session belongs to Alice or someone else. Model performance is evaluated using the ROC-AUC metric.
